Хрестики-Нулики - це гра для двох гравців (Х і О), які по черзі заповняють клітинки на ігровому полі 3 × 3. Гравець, якому вдається розмістити три відповідні позначки в горизонтальному, вертикальному або діагональному рядку (NW-SE та NE-SW) виграє гру. Якщо ти вже розв'язав "Xs and Os Referee", то тобі буде легше розв'язати цю місію.

У цій місії ти будеш грати проти різних ботів, і твоя мета - ніколи не програти. Зараховується лише перемога або нічия. Для кожної гри твоя функція буде викликатися кілька разів (в одному середовищі), для кожного кроку, поки гра не закінчиться.

Для кожного кроку твоя функція отримує стан поля як послідовність рядків, де "X" для хрестика, "O" для нулика і "." для порожньої клітинки. Другий аргумент це твоя позначка, "X" або "O". Для кожного ходу твоя функція повинна повернути координати наступної позначки як послідовність двох чисел - рядка та стовпця. Рядки та стовпці пронумеровані, починаючи з нуля.

Вхідні дані: Два аргументи. Ігрове поле як кортеж рядків (unicode) і твоя позначка як рядок "X" або "O".

Вихідні дані: Координати твого ходу як список або кортеж з двох цілих чисел від 0 до 2.
